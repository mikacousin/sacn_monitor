# sACN Monitor

Simple application to monitor sACN universes.

![Screenshot](../assets/sacn_monitor.png?raw=true)

## Depends on

- python3
- Gtk
- [sACN](https://github.com/Hundemeier/sacn) module

## Usage

```bash
$ git clone https://github.com/mikacousin/sacn_monitor.git
$ cd sacn_monitor
$ python monitor.py
```
